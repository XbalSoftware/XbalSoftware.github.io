EYEbrary User Manual

EYEbrary allows you to build a reusable clinical library and generate clean, patient-facing reports in seconds.

---

1. New Report (Core Workflow)

Main report-building interface

Basic flow

  • Select entries from the sidebar
  • They appear in the report
  • Edit content as needed
  • Export as PDF

Create custom entries

  • Tap the + button (top right)
  • Free-type new content
---

2. Editing Entries

Expanded content editing mode
  • Tap an entry to edit
  • Tap Expand Content for full-screen editing
  • Active entry is highlighted
Use Expand Content for long or detailed edits — it is much more reliable than inline editing.
---

3. Collapsed vs Expanded View

Collapsed view (clean overview)
  • Collapse entries for a quick overview
  • Expand only when editing
---

4. Library (Content Management)

  • Create, edit, and delete entries
  • Assign categories
  • Mark favorites
  • Search instantly
---

5. Entry Editing

  • Edit title and content
  • Toggle visibility in New Report
  • Assign category
  • Mark as favorite
---

6. Batch Select (Bulk Actions)

  • Select multiple entries
  • Apply bulk actions:
    • Delete
    • Export
    • Recategorize
---

7. Categories

  • Create and reorder categories
  • Delete categories (entries fall back to General)
  • Controls filtering throughout the app
---

8. Libraries

  • Multiple libraries supported
  • Switch active library instantly
  • Rename, delete, reorder
---

9. Importing Libraries

  • Merge → adds/updates entries
  • Replace → overwrites library
  • New Library → creates separate library
Disable “Normalize text on import” if formatting is already correct.
---

10. Settings

  • Manage libraries
  • Edit categories
  • Import letterhead
  • Reset app
---

11. Empty / Starting State

  • Select entries from sidebar to begin
---

12. Privacy

  • Patient names are optional
  • Patient names are NOT stored
  • History contains no identifying data
  • Draft restoration excludes patient names
---

13. Tips

  • Use Expand Content for reliable editing
  • Use Batch Select for cleanup and organization
  • Use favorites to speed up workflow
  • Use ⌘Z for undo (keyboard)
---

14. Disclaimer

EYEbrary is a report-generation tool. All content must be reviewed by the practitioner before sharing with patients.
