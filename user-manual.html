EYEbrary User Manual

EYEbrary is a clinical documentation and patient-education tool designed to help eye care professionals quickly generate clear, polished patient reports from a reusable library of content.

Overview

  • Create and maintain your own library of clinical explanations and instructions
  • Quickly assemble reports during or after patient encounters
  • Export clean, patient-friendly PDFs
  • No patient data is stored for privacy
---

Main Interface

New Report screen

Key areas

  • Sidebar: Browse your library
  • Main panel: Build and edit the report
  • Top bar: Switch between New Report, Library, and Settings
---

Creating a Report

1. Add entries

  • Select entries from the sidebar
  • They will appear in the report

2. Create custom content

  • Tap the + button in the top right
  • Free-type a custom entry

3. Edit content

  • Tap any entry to modify title or content
  • Tap Expand Content for full-screen editing

4. Export

  • Tap Export PDF
  • Share, print, or save
---

Library

Library tab

What you can do

  • Create new entries
  • Edit or delete entries
  • Assign categories
  • Mark favorites
  • Search and sort
Use favorites to surface commonly used entries in busy clinic workflows.
---

Batch Select

Batch Select mode
  • Select multiple entries at once
  • Bulk assign categories
  • Delete or export multiple entries
---

Categories

Category manager
  • Organize entries into clinical groupings
  • Reorder categories to match your workflow
  • Delete categories (entries fall back to General)
---

Libraries

Library manager

Features

  • Multiple libraries supported
  • Switch active library instantly
  • Rename, reorder, or delete libraries
---

Importing Libraries

Import options

Import options

  • Merge into Active Library — adds or updates entries
  • Replace Active Library — fully replaces content
  • Import as New Library — creates a separate library
Disable “Normalize text on import” if your library formatting is already correct.
---

Settings

Settings screen
  • Manage libraries
  • Edit categories
  • Import letterhead
  • Reset app
---

History

Recent reports
  • View recent reports
  • Restore previous reports
  • Patient names are never stored
---

Privacy

  • Patient names are optional and never stored
  • History contains no identifying information
  • Draft restoration excludes patient names
---

Tips

  • Use Expand Content for long entries
  • Use Batch Select for large edits
  • Use ⌘Z to undo with a keyboard
  • Organize categories to match your clinical workflow
